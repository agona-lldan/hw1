Компоненты бывают двух разных типов

  • Функциональные Компоненты

    • Такие компоненты создаются с помощью функции, которую можно экспортировать и использовать в дальнейшем

  • Классовые компоненты

    • Такие компоненты создаются с помощью синтаксиса ES6, а именно: создание классов

Компоненты можно использовать с помощью <Work />, здесь Work - это компонент

В компоненты можно передавать специальные пропсы, например <Work name={"lldan"} />, здесь name - это пропс
